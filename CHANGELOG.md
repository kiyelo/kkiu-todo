# Changelog

## 1.0.0 — interaction parity rebuild
- HTML v18.4.8을 390×844에서 홈, 입력, 끼리, 끼리 관리, 더보기, 검색, 완료 시트 상태로 다시 자동 테스트했습니다.
- React판을 동일 상태로 교차 테스트하고 초기 데이터, 멤버, 할 일, 안읽음 수, 광고 행, 완료 행을 맞췄습니다.
- 플로팅 큐의 카드·삽입 슬롯 위치와 긴 제목의 본문/보조문 분리를 목업과 맞췄습니다.
- 끼리 관리에 초대 코드, 끼리/프로필 편집, 10명 멤버 현황, 나가기/삭제 요소를 복원했습니다.
- 더보기에 계정, 알림, 언어, 약관, 문의, 백업/복원, 초기화, 테스트 도구, 버전 이력을 복원했습니다.
- iframe이나 원본 HTML 실행 없이 React 컴포넌트와 Supabase 데이터 계층을 유지합니다.
