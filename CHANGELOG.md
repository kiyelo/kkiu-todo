# Changelog

## 1.3.2 — v18.4.8 터치·슬롯 동등성 복원
- 6점 순서 핸들과 170ms 활성화 지연, 8–9px 이동 취소 조건을 원본대로 복원했습니다.
- 카드 선택 롱탭을 460ms로 맞추고 세로 이동 시 전역 포인터 기준으로 즉시 취소합니다.
- 핸들 재정렬 중 선택 카드 리프트와 주변 카드 실시간 자리 비우기 표현을 추가했습니다.
- 상·하단 앱바를 터치 pointerdown에서 즉시 실행하고 후속 합성 클릭을 차단합니다.
- 더보기에 공용 가변 높이 슬롯·GAP 이동 구조를 적용하고 pair/triple 매치 효과를 연결했습니다.
- 새 할 일 추가 후 삽입 위치를 다음 슬롯으로 이동해 새 카드가 입력 UI 위에 표시됩니다.
- 겹침이 잦은 슬롯·더보기·시트 표면의 불필요한 투명도를 낮췄습니다.
- 끼리·프로필 설정을 원본 모달 구조로 복원해 랜덤·저장 액션을 하단에 고정했습니다.

## 1.3.1 — 모바일 큐 터치 드래그 보강
- 할 일 카드와 더보기 메뉴 항목 위에서도 상하 큐 드래그가 시작되도록 수정했습니다.
- 5px 이동 임계값 전에는 기존 탭·버튼 동작을 유지하고, 드래그가 시작되면 후속 클릭을 차단합니다.
- 드래그 중 가장 가까운 가변 슬롯을 실시간 계산해 입력 카드와 더보기 슬롯이 항목 사이로 즉시 이동합니다.
- 더보기 메뉴도 실제 가변 높이 위치 배열을 사용하고 드래그 중 전환 애니메이션을 해제했습니다.
- 카드 위 드래그, 더보기 항목 위 드래그, 전체 핵심 회귀를 통과했습니다.

## 1.3.0 — HTML 의도 기반 동등성 보강
- 가변 높이 큐 슬롯 모델과 전역 삽입 순번을 적용했습니다.
- 끼리 담당자 필터의 ghost, bundle, 펼치기·접기를 구현했습니다.
- 검색어 강조와 검색 결과 카드 2.7초 강조를 복원했습니다.
- 300ms 완료 애니메이션과 완료 당시 순번 복원을 구현했습니다.
- 오늘·어제·이번 주·이전 완료 그룹과 완료 source unread를 복원했습니다.
- 선택 전용 목록 화면, 프로필 편집 취소 의미, 이모지 한·영 키워드 검색을 복원했습니다.
- 탭·끼리·필터·큐 위치 재실행 영속성과 650ms 읽음 처리를 연결했습니다.
- 개설자 전용 삭제 UI, 멤버 실시간 재정렬, 더보기 슬롯 자동 롤을 보강했습니다.
- 초기 담당자 데이터, 동적 제목 분리, 다국어 광고·입력 문구, 손상된 한글을 수정했습니다.
- Supabase 완료 순번·읽음 영수증·RPC 마이그레이션을 보강했습니다.
- HTML oracle 기반 핵심 통합 회귀에서 런타임 오류 0건을 확인했습니다.
