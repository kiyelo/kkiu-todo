# 끼우 변경 기록

## v18.2.21 — 2026-07-19

- 끼리 검색 버튼을 터치 종료 이벤트로도 직접 처리해 모바일에서 검색 모드 진입이 누락되지 않도록 수정했습니다.

## v18.2.20 — 2026-07-19

- 끼리 설정 목록의 긴 라벨을 끼리·내 프로필로 단순화하고 이름·이모지·찾기 라벨을 입력 영역 위에 정리했습니다.

## v18.2.19 — 2026-07-19

- 끼리 검색 버튼에 직접 진입·닫기 동작을 연결해 검색 화면 전환이 막히지 않도록 보완했습니다.

## v18.2.18 — 2026-07-19

- 더보기 메뉴 카드 사이를 이동하는 전용 플로팅 UI를 복원하고 할일 큐 입력과 분리했습니다.

## v18.2.17 — 2026-07-19

- 긴 끼리 타이틀의 실제 표시 폭을 레이아웃·폰트 로드 후 다시 계산해 전체 텍스트가 옆으로 흐르도록 보완했습니다.

## v18.2.16 — 2026-07-19

- 끼우를 기준으로 끼리·더보기 상단 타이틀의 글자 크기·굵기·줄 높이를 통일했습니다.

## v18.2.15 — 2026-07-19

- 더보기를 할일 큐·플로팅 구조에서 분리하고 메뉴 카드 전용 화면으로 단순화했습니다.

## v18.2.14 — 2026-07-19

- 끼리 검색 버튼을 누르면 전용 검색 화면으로 확실히 진입하고, 상단 버튼에서 페이지 전환 제스처가 시작되지 않도록 보완했습니다.

## v18.2.13 — 2026-07-19

- 끼리 담당자 필터를 좌우로 스크롤하는 동안 페이지 전환 제스처가 시작되지 않도록 터치 영역을 분리했습니다.

## v18.2.12 — 2026-07-19

- 더보기 화면을 계정·환경·데이터 섹션 중심으로 전면 재구축하고 끼우·끼리와 같은 플로팅 삽입 구조로 정리했습니다.

## v18.2.11 — 2026-07-19

- 끼리·프로필 편집창의 이름과 이모지 라벨을 간결한 카드형 구조로 정리하고 저장 동작을 명확하게 개선했습니다.

## v18.2.10 — 2026-07-19

- 끼리 상단 이모지를 고정하고 긴 그룹명이 말줄임표 없이 전체 흐름 표시되도록 개선했습니다.

## v18.2.9 — 2026-07-19

- 끼리 이름·대표 이모지와 내 프로필 이름·이모지를 각각 한 화면에서 함께 편집하도록 통합했습니다.
- 빠른 선택 이모지 칩을 제거하고 대량 이모지 목록·검색·랜덤 선택을 제공했습니다.

## v18.2.8 — 2026-07-19

- 끼우·끼리·더보기 상단을 동일한 높이와 제목 기준선으로 통일했습니다.

## v18.2.7 — 2026-07-19

- 상단 버튼 공통 스타일이 다른 보정 규칙에 덮어쓰이지 않도록 최종 정렬·간격·배경을 고정했습니다.

## v18.2.6 — 2026-07-19

- 끼우와 끼리의 상단 버튼 크기·모서리·간격·아이콘 정렬을 공통 스타일로 통일했습니다.

## v18.2.5 — 2026-07-19

- 끼리 상단을 끼우와 같은 한 줄 구조로 통일했습니다.
- 그룹명이 길어도 두 줄로 늘어나지 않고 흐름 표시를 유지하도록 보완했습니다.

## v18.2.4 — 2026-07-19

- 할 일 수정 중 취소 버튼과 Escape 취소를 추가했습니다.
- 다른 UI를 누르면 수정 모드가 자동으로 해제되도록 수정했습니다.

## v18.2.3 — 2026-07-19

- 끼리 설정에서 전체 이모지 선택 창으로 들어가는 버튼을 명확히 표시했습니다.
- 더보기 플로팅 요소가 메뉴의 첫 위치와 마지막 위치까지 이동하도록 수정했습니다.
- 할 일 수정 완료 뒤 현재 보고 있던 스크롤 위치가 유지되도록 보완했습니다.

## v18.2.2 — 2026-07-19

- 낮은 모바일 화면에서 상단 제목이 잘리지 않도록 화면 높이 계산을 수정했습니다.
- 끼리 제목과 우측 아이콘의 터치 영역을 분리해 설정 버튼이 정상 동작하도록 정리했습니다.
- 긴 끼리 이름의 흐름 표시와 수정 이력의 명시적 닫기 버튼을 추가했습니다.

## v18.2.1 — 2026-07-19

- 끼리 대표·프로필 이모지를 위한 대량 이모지 선택기, 검색, 랜덤 선택을 추가했습니다.

## v18.2.0 — 2026-07-19

- 상단 UI를 고정 크기로 통일하고, 긴 끼리 이름이 흐르도록 보완했습니다.
- 입력 카드 바깥 탭 취소, 롱탭 선택 깜빡임 제거, 더보기 플로팅 구조 통합을 반영했습니다.

## v18.1.2 — 2026-07-19

- 긴 할 일 카드의 첫 줄에서 말줄임표가 다시 생기지 않도록 최종 보정했습니다.

## v18.1.1 — 2026-07-19

- 끼리의 긴 그룹명이 잘리지 않고 최대 두 줄까지 보이도록 상단 UI를 정리했습니다.
- 끼우·끼리·더보기 상단의 제목 기준선과 완료 목록 체크 아이콘 정렬을 보완했습니다.
- 긴 할 일 제목의 첫 줄에 말줄임표가 생길 가능성을 줄였습니다.

## v18.1.0 — 2026-07-19

- 더보기 최하단에 현재 버전과 수정 이력 버튼을 추가했습니다.
- 버전 버튼을 누르면 앱 안에서 간략한 변경 이력을 확인할 수 있습니다.

## v18.0.0 — 2026-07-19

### 화면과 사용성

- 끼우·끼리·더보기의 상단 화면 톤과 버튼 스타일을 정리했습니다.
- 긴 할 일 제목을 더 자연스럽게 표시하고, 완료 목록에서는 카드를 눌러 전체 제목을 펼칠 수 있게 했습니다.
- 검색 결과에서 완료된 할 일을 열어도 검색 화면을 유지하도록 수정했습니다.
- 검색 결과의 미완료 할 일은 입력 카드 바로 위로 이동하고, 잠깐 강조 표시됩니다.
- 끼우와 끼리에서 할 일을 추가하면 입력 카드를 자동으로 닫도록 했고, 입력 중에도 닫기 버튼을 추가했습니다.
- 더보기의 플로팅 요소가 메뉴의 가장 위 위치까지 이동하도록 보완했습니다.

### 이전 v18 개선 포함

- 끼리별 할 일·완료 목록 분리, 한 명 담당자, 담당자 필터와 읽지 않은 업데이트 표시를 정리했습니다.
- 끼리별 내 이름·이모지 및 랜덤 이모지 설정을 추가했습니다.
- 광고·제휴 관련 메뉴를 더보기에서 제거하고, 카드 사이 광고 구조를 위한 화면 흐름을 정리했습니다.
