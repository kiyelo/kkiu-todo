# Changelog

## 1.3.0 — HTML 의도 기반 동등성 보강
- 가변 높이 큐 슬롯 모델과 전역 삽입 순번을 적용했습니다.
- 끼리 담당자 필터의 ghost, bundle, 펼치기·접기를 구현했습니다.
- 검색어 강조와 검색 결과 카드 2.7초 강조를 복원했습니다.
- 300ms 완료 애니메이션과 완료 당시 순번 복원을 구현했습니다.
- 오늘·어제·이번 주·이전 완료 그룹과 완료 source unread를 복원했습니다.
- 선택 전용 목록 화면, 프로필 편집 취소 의미, 이모지 한·영 키워드 검색을 복원했습니다.
- 탭·끼리·필터·큐 위치 재실행 영속성과 650ms 읽음 처리를 연결했습니다.
- 개설자 전용 삭제 UI, 멤버 실시간 재정렬, 더보기 슬롯 자동 롤을 보강했습니다.
- 초기 담당자 데이터, 동적 제목 분리, 다국어 광고·입력 문구, 손상된 한글을 수정했습니다.
- Supabase 완료 순번·읽음 영수증·RPC 마이그레이션을 보강했습니다.
- HTML oracle 기반 핵심 통합 회귀에서 런타임 오류 0건을 확인했습니다.
