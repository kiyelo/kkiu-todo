# 적용 방법

이 ZIP은 React 0.7.0 수정본입니다. GitHub 웹에서 기존 저장소 파일을 하나씩 바꾸지 말고, 가능하면 GitHub Desktop 또는 Codespaces에서 전체 폴더를 반영하세요.

## GitHub Desktop
1. 저장소를 GitHub Desktop으로 연다.
2. 이 ZIP을 푼 뒤 안의 파일 전체를 로컬 저장소 폴더에 덮어쓴다.
3. 변경 목록에서 `.env`나 비밀키가 포함되지 않았는지 확인한다.
4. Summary에 `feat: restore v18.4.8 React parity`를 입력한다.
5. `Commit to main` → `Push origin`을 누른다.
6. GitHub 저장소의 Actions에서 `Deploy Kkiu React to GitHub Pages` 성공을 확인한다.

## 검수
- GitHub Pages에서 끼우·끼리·더보기 탭을 각각 연다.
- 카드와 입력 바가 겹치지 않는지 확인한다.
- 세로 스와이프로 끼우기 위치가 바뀌는지 확인한다.
- 끼리 담당자 필터와 더보기 메뉴 이동을 확인한다.
