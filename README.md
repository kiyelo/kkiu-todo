# 끼우 React 0.9.0

v18.4.8 HTML 목업의 디자인 토큰과 레이아웃 CSS를 React 컴포넌트에 직접 이식한 앱입니다.

- iframe 및 HTML 실행 래퍼를 사용하지 않습니다.
- React 상태·컴포넌트·훅 구조를 유지합니다.
- Supabase 인증과 개인/끼리 데이터 계층을 유지합니다.
- 목업 원본에서 추출한 `mockup-react.css`를 디자인 기준으로 사용합니다.
- 플로팅 큐, 끼우기 슬롯, 끼우·끼리·더보기, 검색, 완료, 수정, 정렬, 담당자와 설정 시트를 React로 구현합니다.

`main` 브랜치에 반영하면 GitHub Actions가 빌드·배포합니다.
