# 끼우

할 일을 원하는 순서에 끼워 넣는 모바일 투두 앱입니다.

- `app/`: React 전환판 소스
- `kkiu4-v18-ux-fixes.html`: v18.4.8 UX 기준 목업
- 루트 `index.html`과 `assets/`: GitHub Pages용 React 빌드

현재 단계는 React 전환판 0.6.0입니다. Supabase 이메일 회원가입·로그인, 개인 할 일과 끼리·끼리 할 일 동기화가 연결되어 있으며, 네 개 데이터 테이블에 RLS 보안 정책이 적용되어 있습니다. 인증 메일은 GitHub Pages의 앱 루트로 돌아오며 잘못된 하위 경로도 404 복구 페이지가 처리합니다. 초대 코드는 다음 백엔드 단계에서 연결합니다.
